1. Based on Mode of Propagation

Single-Mode Fiber (SMF)

  • Uses only one light path
  • Core diameter: ~8–10 µm
  • Supports long-distance communication
  • Very high bandwidth

Used in:

  • Telecom networks
  • Internet backbone
  • Long-distance data transmission

Advantage: Very low signal loss

Disadvantage: Costly, complex equipment

Multi-Mode Fiber (MMF)

  • Allows multiple light paths
  • Core diameter: 50 or 62.5 µm
  • Used for short distances

Used in:

  • LANs
  • Data centers
  • Campus networks

Advantage: Cheaper, easier to install

Disadvantage: More signal dispersion
