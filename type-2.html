2. Based on Refractive Index Profile

Step-Index Fiber

  • Abrupt change in refractive index between core and cladding
  • Light reflects in zig-zag paths

Types:

  • Single-mode step index
  • Multimode step index

Used in: Simple, low-cost systems

Graded-Index Fiber

  • Refractive index gradually decreases from center to edge
  • Light follows a curved path
  • Reduces signal distortion (modal dispersion)

Used in: High-speed LANs
